# 🎮 Gaming Character Accordion UI

Built this modern Gaming Character Accordion UI using HTML, CSS & JavaScript 💻🔥

A clean and interactive accordion component where clicking on a character expands it smoothly while automatically collapsing the others.

---

## 🚀 Features

✔ Smooth Click Animation  
✔ Auto Collapse Effect  
✔ Responsive Design  
✔ Clean UI Interaction  
✔ Organized & Beginner Friendly Code  

---

## 🛠 Tech Stack

- HTML5  
- CSS3 (Flexbox, Transitions)  
- JavaScript (DOM Manipulation)  

---

## 📂 How to Use

1. Clone or download this repository  
2. Open `index.html` in your browser  
3. Customize styles or content as needed
